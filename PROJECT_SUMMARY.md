## 🚀 如何使用

### 启动系统
```bash
# 1. 启动后端
cd backend
python api_server.py

# 2. 启动前端
cd web
python -m http.server 8080
```

### 访问地址
- 前端: http://localhost:8080
- 后端: http://localhost:8000
- API 文档: http://localhost:8000/docs
